# Competency Radar Chart Study

โปรเจกต์นี้เป็นการศึกษาการใช้งาน React ร่วมกับ MUI X Radar Chart
เพื่อแสดงผลข้อมูลสมรรถนะ (Competency Profile) ของผู้เรียน
ซึ่งเป็นส่วนหนึ่งของการพัฒนาโครงงานระบบวิเคราะห์สมรรถนะผู้เรียน

## Objectives
- ศึกษาการใช้งาน Radar Chart ด้วย MUI X Charts
- ทดลองแสดงข้อมูลสมรรถนะหลายมิติ
- เตรียมความพร้อมสำหรับนำไปใช้ในระบบจริง
